# Thuật ngữ dự án (Glossary)

> Danh sách thuật ngữ, viết tắt, và định nghĩa được sử dụng trong toàn bộ tài liệu dự án.
> CẬP NHẬT mỗi khi xuất hiện thuật ngữ mới trong bất kỳ tài liệu nào.

---

## Quy ước

- Sắp xếp theo alphabet
- Mỗi thuật ngữ có: Tiếng Việt, Tiếng Anh (nếu có), Định nghĩa, Xuất hiện lần đầu ở tài liệu nào
- Viết tắt liệt kê riêng ở cuối

---

## Thuật ngữ

| Thuật ngữ (VN) | English | Định nghĩa | Xuất hiện tại |
|---|---|---|---|
| _[Ví dụ: Yêu cầu chức năng]_ | _[Functional Requirement]_ | _[Yêu cầu mô tả hành vi cụ thể mà hệ thống phải thực hiện]_ | _[SRS]_ |

---

## Viết tắt

| Viết tắt | Đầy đủ | Giải thích |
|---|---|---|
| PRD | Product Requirements Document | Tài liệu yêu cầu sản phẩm |
| SRS | Software Requirements Specification | Đặc tả yêu cầu phần mềm |
| TDD | Technical Design Document | Tài liệu thiết kế kỹ thuật |
| ADR | Architecture Decision Record | Bản ghi quyết định kiến trúc |
| SDLC | Software Development Life Cycle | Vòng đời phát triển phần mềm |
| MoSCoW | Must/Should/Could/Won't | Phương pháp ưu tiên yêu cầu |
| FR | Functional Requirement | Yêu cầu chức năng |
| NFR | Non-Functional Requirement | Yêu cầu phi chức năng |
| IR | Interface Requirement | Yêu cầu giao diện |
| DR | Data Requirement | Yêu cầu dữ liệu |
| AC | Acceptance Criteria | Tiêu chí chấp nhận |
| WCAG | Web Content Accessibility Guidelines | Hướng dẫn truy cập nội dung web |
